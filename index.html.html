Connect Google Calendar

1
Create a Google Cloud project & enable Calendar API

Go to console.cloud.google.com → New Project → APIs & Services → Enable "Google Calendar API"

2
Create an OAuth 2.0 Client ID

APIs & Services → Credentials → Create Credentials → OAuth Client ID → Web Application. Add http://localhost and the URL where you host this file to Authorized JS Origins.

3
Add your redirect URI in Google Console

In your OAuth Client ID settings → Authorized Redirect URIs → add the exact path to this HTML file. If opening locally use file:///C:/path/to/office-calendar.html. If hosted on a server use that URL instead.

4
Share all business calendars to one primary Google account

In each business Google account → Calendar Settings → Share with specific people → add your primary account email with "See all event details".

5
Paste your Client ID below and connect

Clicking Connect will redirect you to Google sign-in, then bring you straight back to the calendar.
